# GitHub + Netlify Otomatik Deploy

Bu proje statik web/PWA olarak `web/` klasorunden yayinlanir.

## GitHub

1. GitHub'da yeni repo olustur.
2. Bu projeyi o repoya yukle.
3. Ana branch olarak `master` veya `main` kullanabilirsin.

## Netlify

1. Netlify'da `Projects` ekranina gir.
2. `Import a Git repository` bolumunden GitHub'i sec.
3. Repo olarak bu projeyi sec.
4. Build command alanini bos birak.
5. Publish directory alanina `web` yaz.
6. Deploy et.

## Sonraki Guncellemeler

Kod GitHub'a push edilince Netlify siteyi otomatik yeniler. Link ayni kalir.
